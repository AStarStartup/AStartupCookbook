## Quickstart

1. Clone the AStarStartup GitHub Organization workspace.

```BASH
git clone https://github.com/AStarStartup/AStartStartup.git --recursive
```

```BASH
git clone https://github.com/CookingWithCale/AStartupCookbook.git
```
